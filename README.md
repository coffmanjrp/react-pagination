## How to start

1. Clone this repository.

```
git clone git@github.com:coffmanjrp/react-pagination.git
```

Or download the zip file.

2. Run following command to install the dependencies.

```
npm install
# or
yarn
```

3. And you can start localhost by

```
npm start
# or
yarn start
```

Enjoy!
